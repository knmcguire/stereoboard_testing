----------------------------------------------------------

data file formats:


'position.dat'

 - timestamp of logging position data
 - timestamp from Optitrack system
 - X location 	[m]
 - Y location 	[m]  (vertical!)
 - Z location 	[m]
 - roll angle 	[deg]
 - pitch angle 	[deg]
 - yaw angle 	[deg]  (wrt negative-Z-axis)


'timing.dat'

 - timestamp of logging image
 - constant number 1
